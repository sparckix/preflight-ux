# Launch Surface

Apple Maps was presented as the built-in default maps experience in iOS 6.
Users opened the Maps app for search, location lookup, route planning, and
navigation.

Reported launch-surface characteristics:

- Apple Maps replaced the previously trusted Google Maps default
- users relied on map search, landmarks, routing, and transit information
- the launch version contained missing, misplaced, or inaccurate locations and
  landmarks
- public transit directions were absent compared with the prior default
- Apple recommended third-party or web-map alternatives while improving Maps

The benchmark should be run as a trust/data-quality case for a default utility
app with real-world consequences.
