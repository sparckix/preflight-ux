# Launch Surface

The redesigned Snapchat app separated social content from media content. The
Friends page combined incoming Snaps, chats, and friend Stories in one place,
with friend content no longer presented in the previous familiar chronological
Story list. Discover/media content lived separately.

Reported surface characteristics:

- friend Stories appeared interleaved with private messaging rather than in the
  previous Stories area
- the Friends page was algorithmically sorted rather than simply chronological
- the send flow made finding friends feel harder for existing users
- rewatching Stories required more effort than in the prior interface
- users could not revert to the previous version once updated

The benchmark should be run as a redesign/regression case: personas should be
told that this is an existing social app with habitual daily use and a changed
navigation model.
