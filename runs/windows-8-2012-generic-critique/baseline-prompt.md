## Product Surface

# Launch Surface

The Windows 8 launch surface combined two environments:

- a tile-based Start screen optimized for touch
- a traditional desktop environment for legacy PC work
- hidden Charms for generic commands such as Search, Share, and Settings
- Windows Store apps that emphasized full-screen, low-density layouts
- reduced visible multi-window affordances compared with prior desktop Windows

The benchmark should be run as an operating-system interaction case. Personas
should know the product serves both novice users and experienced PC users, many
of whom have existing Windows habits.
