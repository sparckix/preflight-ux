Brief Surface Panel Review

How to use

Bring the product surface you want reviewed, choose the panel, then run it with your own model key. Findings are hypotheses to validate, not user-research proof.

  1. Describe the job.Name the product, audience, stage, and what a user is trying to do.
  2. Add the surface.Paste the spec or copy, add a prototype URL, and upload screenshots for visual review.
  3. Select risks.Pick personas, issue classes, and review depth for the decision you are stress-testing.
  4. Run or export.Run with BYOK, download a review brief JSON, or export a repo-ready Markdown report.
